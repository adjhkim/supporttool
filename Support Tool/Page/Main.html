공지사항
문서 기본 사용법
▼
1. 상단에 있는 메뉴바를 통해 원하는 페이지로 이동이 가능합니다.

2. 메뉴바에 있는 로고를 클릭하면 첫 화면으로 돌아옵니다.

3. 각 페이지별로 제공하는 기능은 다음과 같습니다.
 - 문의 카테고리 분석 : 월별 데이터를 표 형식으로 출력, 1~3순위 카테고리 표시, 행(row) 클릭 시 그래프 팝업
 - 우편 문구 생성 : 입력된 채팅 신고 로그를 기반으로 제재 정보와 채팅 내용을 자동으로 우편 문구에 담아 템플릿 생성
 - HTML 태그 생성 : 에디터 소스코드란에 붙여 넣을 수 있는 HTML 태그를 생성, 상자/표형식 2가지 디자인 지원
 - 게임 로그 분석 보조 : 로그 데이터를 원하는 타입만 골라 필터링, 특정 파라미터별로 필터링
안내드립니다.
